# tech-stocks-factor-model
This project analyzes how the rise of generative AI (2022-2023) altered the explanatory power of traditional asset-pricing factors for selected U.S. technology stocks. Using the Famaâ€“French Five-Factor Model, a model describing stock returns,  we compare factor significance and explanatory strength before and during the AI-driven market boom.
